Installation

Installing Fastone Template

Fastone Setup (React)

  • Make sure to install nodejs and npm to run this template. You can download them by the following link below
    https://nodejs.org
    https://docs.npmjs.com/cli/v8/commands/npm-install.
    -Please, note that we recommend the LTS(Long term support version for the Node.js)
  • Optional - GitBash
    https://gitforwindows.org/

Npm Package install process

  • You must have to install packages. You can do this by running 'npm install' from the root of your project to install all the necessary dependencies.
  • --To install them, first, open up your project directory and then 'right click' of your mouse after that click on 'Git Bash Here'
    --It will open up the terminal
    --Now type 'npm'

Development server

  • -When the package install process is finished, run 'npm start' from the root of your project to start the dev server:-
    --Navigate to http://localhost:3000/
    --The app will automatically reload if you change any of the source files.

Build

  • Run 'npm build' to build the project. The build artefacts will be stored in the build/ directory
